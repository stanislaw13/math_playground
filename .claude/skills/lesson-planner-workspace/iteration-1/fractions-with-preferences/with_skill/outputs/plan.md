# Fractions Lesson Plan for Primary School

## Overview
An interactive lesson introducing students to fractions through visual metaphors and hands-on activities. The lesson progresses from understanding parts of a whole to recognizing equivalent fractions.

## Learning Objectives
By the end of this lesson, students will be able to:
1. Identify and name fractions using visual representations
2. Understand fractions as parts of a whole
3. Recognize equivalent fractions
4. Match fractions to their decimal equivalents
5. Build confidence in working with fractions

## Lesson Structure

### Section 1: Theory - Introduction to Fractions
**Interactive Fraction Bar Explorer**
- A horizontal bar divided into sections that students can slide from 1/1 (whole) to smaller fractions
- As students slide, the fraction label updates (1/1 → 1/2 → 1/3 → 1/4, etc.)
- Shows the equivalent decimal representation alongside (0.5, 0.33, 0.25, etc.)
- Interactive shading shows the fraction amount with color
- Students learn that fractions represent parts of a whole

**Visual Metaphor**
- KaTeX formulas showing: numerator/denominator = parts/whole
- Emphasize that fractions are division: a ÷ b

### Section 2: Pizza Slices Game - Visual Pizza/Pie Game
**Game Type:** New interactive visual game
**Skill Categories:**
- `pizza-halves`: Understanding halves (1/2)
- `pizza-thirds`: Understanding thirds (1/3)
- `pizza-fourths`: Understanding fourths (1/4)
- `pizza-eighths`: Understanding eighths (1/8)

**Mechanics:**
1. Student sees a pizza or pie divided into sections
2. A target fraction is shown (e.g., "Select 3/4 of the pizza")
3. Student clicks on slices to select them
4. Feedback shows if the correct amount is selected
5. Progressive difficulty: start with halves/thirds, advance to more complex divisions

**Adaptive Features:**
- Uses adaptive engine to track which fraction types students struggle with
- Adjusts how many questions to ask about each fraction type
- Promotes difficulty when student shows mastery

### Section 3: Match Pairs Game - Fraction to Decimal Equivalents
**Game Type:** Reusing MatchPairsGame component
**Skill Categories:**
- `fraction-decimal-simple`: Simple fractions (1/2, 1/4, 1/5)
- `fraction-decimal-complex`: Complex fractions (3/4, 2/5, 3/8)
- `fraction-decimal-comparison`: Comparing fraction sizes

**Pairs Generated:**
- Left side: Fraction notation (1/2, 3/4, 2/5, etc.)
- Right side: Decimal equivalents (0.5, 0.75, 0.4, etc.)
- 6-8 pairs per round
- Cards shuffled and students match

**Adaptive Features:**
- Pair generator accepts difficulty parameter
- Easy: common fractions (1/2, 1/4, 1/5)
- Medium: mixed fractions (3/4, 2/5, 3/8)
- Hard: less common fractions (5/8, 7/10, 4/9)

### Section 4: Bonus - Fraction Size Comparison Challenge
**Game Type:** Multiple choice or ordering game
**Mechanics:**
- Show 2-3 fractions and ask which is larger
- Or show 4 fractions and ask to order from smallest to largest
- Uses visual representation alongside notation

**Adaptive Features:**
- Tracks confidence with fraction comparison
- Difficulty increases with more fractions to compare and less familiar fractions

## Formulas & Definitions
- Fraction = numerator ÷ denominator (parts ÷ whole)
- Equivalent fractions: a/b = (a×n)/(b×n)
- Fraction to decimal: a ÷ b

## Assessment
- Games track student accuracy across skill categories
- Students see summary of weaker areas to practice
- Progression through adaptive difficulty ensures appropriate challenge level
